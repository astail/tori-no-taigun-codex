# 参考調査

調査日: 2026-07-16

## 確認した要素

- 任天堂の公式紹介では、シリーズの核が「リズムに合わせて、タイミングよくボタンを押す」単純操作であることを確認した。
- 「鳥の大群 / Flock Step」は一定拍ごとの行進、音の掛け声に続くしゃがみとジャンプ、群れとの同期、隣の鳥にぶつかる失敗表現を持つことを複数資料で確認した。
- 資料上の原作テンポは130 BPMで、行進は1拍ごと。今回の仕様では約1分・50判定・満点100点という要件を同時に満たすため、100 BPMの2拍ごとへ独自に再設計した。
- 原作の画像・動画・音源は実装素材へ取り込まず、動きの役割だけを仕様化した。
